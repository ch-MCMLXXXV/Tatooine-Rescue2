## About The Project



This project showcases the backend and frontend process of building a complete website. You will explore creating your database as well as creating seed data to test it, the api to make the necessary calls, routes to get to the page rendering and front-end api calls for the user to interact with the page.
You will also cover styling using Material-UI but this can be changed to your preference.
